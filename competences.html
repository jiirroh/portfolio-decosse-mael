Bloc 1 - Support et mise à disposition de services informatiques

Ce bloc regroupe les compétences liées à l’installation, la maintenance et l’assistance des services informatiques.

Gestion du parc informatique

Mise en place d’un inventaire automatisé des ressources informatiques (PC, imprimantes, périphériques) avec un script Python et base SQLite. Suivi des mises à jour logicielles et vérification des droits d’accès.

Support utilisateur - Helpdesk

Développement d’une interface web d’assistance (PHP/SQL) permettant de collecter, suivre et traiter les demandes des utilisateurs. Implémentation d’un tableau de bord de suivi des tickets.

Veille technologique automatisée

Mise en place d’un outil de veille automatisée en Python collectant les flux RSS de sources tech (ZDNet, Developpez.com) pour suivre les tendances du web et de la cybersécurité.

Bloc 2 - Conception et développement d’applications

Ce bloc concerne la conception, le développement, les tests et la maintenance d’applications.

Serveur GTA RP - Scripts LUA

Développement complet d’un serveur GTA RP en LUA : gestion d’inventaires, économie, jobs, interface utilisateur et optimisations côté client/serveur. Gestion des permissions et sécurité des scripts.

Application de gestion en Java (POO + MySQL)

Réalisation d’une application de gestion des clients en Java avec architecture MVC. Connexion à une base de données MySQL, validation des formulaires et génération de rapports PDF automatisés.

Site web dynamique (HTML / CSS / JS / PHP)

Création d’un site web complet avec base de données MySQL : interface d’administration, espace utilisateur, design responsive et sécurité (hashage des mots de passe, filtres anti-injection).

Bloc 3 - Cybersécurité des services informatiques

Ce bloc regroupe les compétences liées à la sécurité, la confidentialité et la continuité des services informatiques.

Audit RGPD et protection des données

Réalisation d’un audit de conformité RGPD pour un intranet. Identification des traitements de données personnelles et mise en place de procédures de consentement.

Configuration réseau et pare-feu

Configuration d’un pare-feu sous pfSense, segmentation du réseau (LAN/WAN/VLAN) et définition des règles d’accès utilisateurs. Tests de robustesse via analyse de ports.

Analyse de logs et détection d’intrusion

Mise en place d’un script Python pour l’analyse de logs système (auth.log / syslog) et détection d’activités suspectes. Export automatisé des alertes dans un fichier journal.
